# Python3常用知识要点

### 头部编码设置
```python
# -*- coding: utf-8 -*-
```

### 命令行运行
`python xxx.py`

### 输入输出
* 输出
    ```python
    print('hello, world')

    #遇到逗号“,”会输出一个空格
    print('The quick brown fox', 'jumps over', 'the lazy dog')
    ```
* 输入
    ```python
    name = input()

    name = input('please enter your name: ')
    ```
